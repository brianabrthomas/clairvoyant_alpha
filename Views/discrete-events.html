Discrete Events

Catalysts are not binary — they carry probability, expected magnitude, direction, and resolution windows. An "event" is a statistically abnormal market response that demands explanation. News is searched after the event is detected, never before. This ordering prevents narrative hallucination.

Catalyst Pricing Term Σk γk · E[Eventk] = Σk γk · P(event) · E[magnitude | event]

Capabilities

Poisson / Hawkes Process Modeling
Event arrival timing modeled as Poisson (baseline) or Hawkes (self-exciting) processes. Entity-specific, regime-adjusted, and seasonally calibrated intensity parameters.
Event Study Framework (CAR/CAAR)
Cumulative Abnormal Return computation with market model, sector model, or factor model benchmarks. Estimation and event windows configurable per study.
Conditional Return Distributions
Per event type: mean return path, distribution of outcomes, skew, tail risk, volatility decay, drawdown probability, and time-to-mean-reversion.
Mechanism Attribution
CFA-grade economic mechanism mapping: cashflow revision, discount rate shift, margin pressure, demand shock, policy risk. Ranked by plausibility, timing, and precedent.
Event Taxonomy & Pattern Library
Canonical event type tags reused across history: earnings surprise, guidance cut, regulatory overhang, liquidity squeeze. Each with computed outcome distributions.
Catalyst Risk Pricing
How much catalyst risk is priced into the current stock price? Decomposition of price into fundamental anchor + priced catalysts + residual uncertainty.

Statistical Artifacts (Precomputed)

Event Study Library
Every historical event with CAR/CAAR, abnormal return distribution, estimation window parameters, and regime tag. Immutable once created.
Hawkes Process Parameters
Self-exciting intensity models per entity and event type. Decay rates, baseline intensities, and jump magnitudes estimated via MLE with bootstrap confidence.
Catalyst Expected Value Tables
P(event) x E[magnitude | event] tables per catalyst type and entity. Updated continuously as new data arrives. Direct input to the pricing equation.
